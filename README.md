# RBCProjectMVC

*RBCProjectMVC* is a web-based Employee Management System built with **ASP.NET Core MVC**. The system allows managing employee records, searching, sorting, and exporting data to PDF or Excel. It also supports displaying employee photos and visualizing department statistics with charts.

---

## Features

- *Employee Management*: View, add, edit, and delete employee information.
- *Live Search & Sorting*: Quickly search and sort employees using AJAX.
- *PDF Export*: Export the current employee list to PDF with report details.
- *Chart Visualization*: Display department-wise employee counts using histograms.
- *User Friendly Interface*: Responsive UI with Bootstrap 5 and jQuery.

---

### Technologies

- **.NET 8**: Core framework for the backend.  
- **Entity Framework Core**: ORM for database operations.  
- **SQL Server**: Database management system.  
- **Select.HtmlToPdf.NetCore**: For PDF generation.  
- **Chart.js**: For displaying department statistics.  
- **Bootstrap 5 & jQuery**: Frontend UI and interactivity.
